# Program to Find Max

This post introduces you how you can simply find the max using ternary operator in Java programming.

```java
class Test {
    public static void main(String[] args) {
        int x = 10, y = 20, max;
        max = (x > y) ? x : y;
        System.out.println("Max=" + max);
    }
}
```

```bash
    Max=20
```
